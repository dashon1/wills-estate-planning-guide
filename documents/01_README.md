## Reading Order Recommendation

### For First-Time Users
1. Start with `02_Wills_Estate_Planning_Guide.pdf` - read completely
2. Complete `06_Assets_Inventory_Template.pdf` to catalog your assets
3. Use `03_Legal_Requirements_Checklist.pdf` as your working checklist
4. Review `05_Estate_Planning_Terms_Glossary.pdf` for any unclear terms
5. Read `04_Executor_Duties_Guide.pdf` to understand executor responsibilities

### For Users Updating Existing Wills
1. Review `03_Legal_Requirements_Checklist.pdf` to identify changes needed
2. Update `06_Assets_Inventory_Template.pdf` with current asset information
3. Cross-reference with state-specific requirements if upgrading to Premium

### For Premium Users
1. Complete all Starter tier documents first
2. Review `07_Premium_Bonus_1_State_Specific_Guide.pdf` for your state
3. Complete all advanced directive documents (8-11)
4. Read `09_Premium_Bonus_3_Trust_Fundamentals_Guide.pdf` to evaluate trust options
5. Business owners should review `12_Premium_Bonus_6_Business_Owner_Estate_Planning.pdf`

## Quick Start Checklist

Before diving into detailed documents, complete these steps:

- [ ] Gather identification documents (driver's license, passport)
- [ ] List all financial accounts and approximate balances
- [ ] Identify all real estate holdings with addresses
- [ ] Note beneficiaries on existing accounts (401k, IRA, insurance)
- [ ] Decide who you want as executor of your estate
- [ ] Consider who should be guardian of any minor children
- [ ] Review any existing wills or trust documents
